    Strings

    • En este documento, irá toda la información que pueda documentar de las cadenas de texto en java.
    • El objetivo es que pueda servir de consulta rápida en caso de necesitar sustento en explicación de funcionamiento.
    • Este documento no se considerará completo, a menos que se cierre y se indique su versión.


    Métodos: de una instancia

    • Que sean de una instancia, quiere decir que, estos métodos se aplican a una instancia de new String("foo") o "foo". Es decir que los métodos no son estáticos.

    char charAt(int index):

    • Retorna el carácter del índice dado, comenzando en cero.
    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String numeros = "12345";

        System.out.println(numeros.charAt(0));
    }
}

    OUTPUT: 1

    Ejercicio

    • Crear un programa que cuente las repeticiones de un carácter en una cadena.

    Solución

    • Para el carácter: 123456495678295768273465479, lo recorremos y usando el método, lo comparamos con el carácter en la posición i, aumentando el contador.
    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String str = "123456495678295768273465479";

        // initialized the counter to 0
        int contador = 0;

        for (int i = 0; i < str.length(); i++) {
            if (str.charAt(i) == '2') { // char == no .equals, en el fondo char es un unicode number.
                // incrementamos el número a medida que hay ocurrencias.
                contador++;
            }
        }
        System.out.println("El 2, se repite " + contador + " veces en la cadena");
    }

}

    int compareTo(String string):

    • Se puede usar para ordenar por orden alfabético ya que:

      • == 0 si los Strings son iguales
      • < 0 si el primero es menor
      • > 0 mayor que, > 0 si es mayor.

    • Compara el valor Unicode. Es decir, la diferencia numérica entre estos.

    Ejemplo

    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String str1 = "a";
        String str2 = "b";
        String str3 = "c";

        /**
         * resultado -1, ya que la distancia entre a y b en unicode es 1, pero como a
         * está primero, está una posición antes en unicode por eso es -1, en caso
         * contrario sería 1.
         */
        System.out.println("str1 compareTo str2: " + str1.compareTo(str2));
    }
}
    • Podemos tener también la diferencia de cantidad de caracteres.
    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String str1 = "a";
        String str2 = "abcd";

        /**
         * Podemos ver, que literalmente, str1 tiene -3 caracteres que la segunda
         * cadena.
         * 
         * También podemos ver que str2 posee 3 caracteres más que str1
         */
        System.out.println("str1 compareTo str2: " + str1.compareTo(str2));
        System.out.println("str2 compareTo str1: " + str2.compareTo(str1));

    }

}

    Salida:

    str1 compareTo str2: -3
str2 compareTo str1: 3

    boolean startsWith(String prefix, int offset):

    • Chequea si una cadena comienza con...
    • El offset indica desde donde comenzamos a medir, partiendo que 5, significa que se saltará cinco espacios antes de comenzar a medir.

    Ejemplo

    // tomado de interner, no merece más el ejemplo
package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        // given string
        String s = "This is just a sample string";

        // checking whether the given string starts with "This"
        System.out.println(s.startsWith("This"));

        // checking whether the given string starts with "is" 5 offset apply.
        System.out.println(s.startsWith("is", 5));

    }

}

    Output

    true
true

    boolean startsWith(String prefix):

    • Lo misma función anterior; pero sin el offset.

    boolean endsWith(String suffix)

    • Misma funcionalidad anterior, pero en el final de la cadena, sin offset.

    int indexOf(int ch):

    • Retorna el índice de la primera ocurrencia de un char, que puede ser un String de un carácter.
    • En el ejemplo, se puede ver que es case-sensitive y que el índice comienza en cero.

    Ejemplo

    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String letras = "abdffFF";

        System.out.println(letras.indexOf('F'));

    }

}

    Output

    5

    int lastIndexOf(int ch):

    • Retorna la última ocurrencia de un carácter dado.
    • Índice comienza en cero.

    Ejemplo

    }

}

    Output

    Última b, en: 3

    int lastIndexOf(int ch, int fromIndex):

    • Al igual que el anterior; pero este método comienza a buscar hasta fromIndex
    • En la primera b, dice que la encuentra en la posición 1, pero por que le dijimos que buscara hasta la posición 1, donde ta tenemos la primera B.
    • Ambos índices comienzan en cero.

    Ejemplo

    package cl.lherrera.java001;


public class Principal {

    public static void main(String[] args) {
        String letras = "abcdefgb";

        System.out.println(letras.lastIndexOf('b', 1));
        System.out.println(letras.lastIndexOf('b'));

    }

}

    Output

    1
7

    String substring(int beginIndex, int endIndex):

    • Extrae una sub-cadena, indicando el principio y el fin.
    • El principio está incluido, comienza en cero.
    • El final no se incluye, el que se incluye es el carácter anterior, comienza en cero.
    • Su versión con parámetro de inicio es String substring(int beginIndex). Que solamente indica el inicio.

    Ejemplo

    package cl.lherrera.java001;


public class Principal {

    public static void main(String[] args) {
        String letras = "0123metro9";

        // desde la posición 4 hasta la 8 incluída.
        System.out.println(letras.substring(4, 9));
    }
}

    Output

    metro

    String[] split(String regex, int limit):

    • Separa una cadena de caracteres y los transforma en una cadena
    • Permite incluir un separador y un delimitador, si el delimitador, indica el tamaño en elementos, que tendrá el arreglo resultante.
    • Si se quiere solamente separar por el delimitador, con el total de elementos separados, se utiliza String[] split(String regex), que es un equivalente al delimitador igual a cero. Veamos el comportamiento de este método para comprender mejor su funcionamiento.
    • Dependiendo el delimitador, podemos ver la cantidad de elementos del arreglo, siendo el 0 el que indica sin límite.

    Ejemplo

    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String unidas = "m,e,t,r,o";
        System.out.println(unidas);

        String[] separadas0 = unidas.split(",", 0);
        String[] separadas1 = unidas.split(",", 1);
        String[] separadas2 = unidas.split(",", 2);
        String[] separadas3 = unidas.split(",", 3);

        imprimeArreglo("separadas0", separadas0);
        imprimeArreglo("separadas1", separadas1);
        imprimeArreglo("separadas2", separadas2);
        imprimeArreglo("separadas3", separadas3);

    }

    private static void imprimeArreglo(String nombre, String[] arreglo) {
        System.out.println("\n" + nombre + "\n");

        for (int i = 0; i < arreglo.length; i++)
            System.out.println("Elemento[" + i + "] = " + arreglo[i]);

    }

}

Elemento[4] = o

    }
}

    Output

    String antes trim: [ casa 600 stgo.              ] - fin
String después trim: [casa 600 stgo.] - fin

    Ejercicio propuesto

    • crear un método, que valide un inicio de sesión, comparando las entradas, suprimiendo espacios en blanco y diferencias entre mayúsculas y minúsculas.

    Solución

    package cl.lherrera.java001;

import java.util.Scanner;

public class Principal {
    private static final String USUARIO = "lherrera";
    private static final String PASSWORD = "1234";

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.println("Ingrese el nombre de usuario...");
        String usuario = sc.nextLine();

        System.out.println("Ingrese su contraseña...");
        String password = sc.nextLine();

        if (credencialesOk(usuario, password))
            System.out.println("Credenciales correctas, bienvenido al sistema App !!");
        else
            System.out.println("Credenciales inválidas...");

    }

    private static boolean credencialesOk(String usuario, String password) {
        boolean usuarioOk = usuario.trim().toLowerCase().equals(USUARIO.toLowerCase());

        // Esta información será exacta
        boolean passwordOk = password.equals(PASSWORD);

        return usuarioOk && passwordOk;
    }
}

    Métodos estáticos

    public static String format(String format, Object... args)

    • Utilizado para dar formatos en las cadenas.

    Ejemplo

    package cl.lherrera.java001;

public class Principal {

    public static void main(String[] args) {
        String str = "just a string";  

        //concatenating string using format
        String formattedString = String.format("My String is %s", str);  

        /*formatting the  value passed and concatenating at the same time
         * %.6f is for having 6 digits in the fractional part
         */
        String formattedString2 = String.format("My String is %.6f",12.121);

        System.out.println(formattedString); 
        System.out.println(formattedString2); 
    }
}

    Output

    My String is just a string
My String is 12,121000
    • Se puede ver que se reemplaza %s, por una cadena y %.6f, define a un flotante de 6 dígitos decimales como resultado.
    • Existen más formas de formatos, que deben ser profundizados; en una próxima iteración.

    Otras por definir ejemplos.

    • String concat(String str): Concatena al final del String, como el +.
    • boolean contains(CharSequence s): Verdadero o falso si CharSequence está o no presente en la cadena.
    • String toUpperCase(): Retorna una cadena en mayúsculas de la instancia.
    • String toUpperCase(): Retorna una cadena en minúsculas de la instancia.
    • public boolean isEmpty(): Lo mismo que decir str.lenght() == 0; pero más directo.
    • int length(): Retorna el largo de una cadena.
    • byte[] getBytes(): Mostrado por que se usa para técnicas de cifrado y transporte, además de escritura de archivos, lo que hace, es simplemente obtener los bytes de una cadena de caracteres.
    • boolean matches(String regex): Usado para reconocer secuencias de texto, como identificar fechas en una web. Reconoce si un patrón de una expresión regular.
