    ARREGLOS EN JAVA 0️⃣ - 1️⃣ - 2️⃣ - ...

    • Un array (arreglo) en Java, es una estructura de datos que nos permite almacenar un conjunto de datos de un mismo tipo.
    • El tamaño de los arrays se declara en un primer momento y no puede cambiar.


    ARREGLOS UNIDIMENSIONALES(UNA DIMENSIÓN).

    • Cuando creamos un array de nombre “a” y de tamaño “n” (int[] a = new int[n]) estamos creando n variables que son a[0], a[1], a[2], ..., a[n-1].
    • Los arrays se numeran desde el elemento cero, que sería el primer elemento, hasta el n-1 que sería el último elemento.

    RECORRER UN ARREGLO

    Existen mejores formas de acceder a cada elemento del arreglo utilizando el ciclo for

    String[] autos = {"tesla", "Lexus", "Volvo"};

// Método 1: Recorrer usando for y conociendo el largo del arreglo
System.out.println("Método 1:");
for(int i = 0; i < 3; i++){
    System.out.println(autos[i]);
}

// Método 2 : Recorrer usando for y desconociendo el largo del arreglo.
System.out.println("\nMétodo 2:");
for(int i = 0; i < autos.length; i++){
    System.out.println(autos[i]);
}

// Método 3 (El mejor): Recorrer usando for y desconociendo el largo
// del arreglo. (Foreach)
System.out.println("\nMétodo 3:");
for (String auto : autos) {
    System.out.println(auto);
}

    VARIABLE ARGUMENTS varargs...🤔

    • Desde la 5ta versión del JDK, existe la posibilidad de implementar métodos con parámetros variables utilizando el Variable Arguments, es decir, al momento de llamar a este método entregando una cantidad variable de argumentos
    • Existía la posibilidad de hacerlo con un arreglo, por ejemplo:
    public static void main(String[] args ) {
    //Todo el código acá
}
    • Pero a partir de JDK 5, se puede hacer de la siguiente forma:
    public static void main(String ...args ) {
    //Todo el código acá
}

    Ejemplo: suma de argumentos.

    • Crear un método que reciba una cantidad de indeterminada de parámetros e imprima el resultado de la suma.
    package cl.lherrera.primero;

public class Principal {

    public static void main(String[] args) {
        sumar(1, 2, 3, 4, 5);
    }

    public static void sumar(int... numeros) {
        int suma = 0;

        for (int i = 0; i < numeros.length; i++) {
            suma += numeros[i];
        }

        System.out.print("La suma es: " + suma);
    }
}

    ARRAYS MULTI-DIMENSIONALES

    • En Java es posible crear arreglos con más de una dimensión, pasando de la idea de lista, vector o matriz de una sola fila a la idea de matriz de m x n elementos, estructuras tridimensionales, tetra-dimensionales, etc. La sintaxis será:
    Tipo_de_variable[][]… [] Nombre_del_array = new Tipo_de_variable[dimensión1][dimensión2]…[dimensiónN];

int[][]  matriz = new int[3][2];
O alternativamente
int[][]  matriz;
matriz = new int[3][2];
O directo:
int[][] matriz = {{1,2},{3,4},{5,6}}; //Tres columnas y dos filas.

    // Declaraciones.
byte[][] edad = new byte[4][3];
short ][] edad = new short[4][3];
int[][] edad = new int[4][3];
long[][] edad = new long[4][3];
float[][] estatura = new float[3][2];
double[][] estatura = new double[3][2];
boolean[][] estado = new boolean[5][4];
char[][] sexo = new char[2][1];
String[][] nombre = new String[2][1];

// Asignar valores:
matriz[0][0] = 1; matriz[0][1] = 2; matriz[1][0] = 3; matriz[1][1] = 4; matriz[2][0] = 5; matriz[2][1] = 6;

// Directamente:
int[][] matriz = {{1,2},{3,4},{5,6}}; //Tres columnas y dos filas.

// Recorrer un arreglo multi-dimensional
int[][] matriz = {{1,2},{3,4},{5,6}}; // tres columnas, dos filas [3][2]

// Método 1: Conociendo exactamente el número de filas y columnas.
System.out.println("Método 1");
for(int i = 0; i < 3; i++){
    for(int j = 0; j < 2; j ++){
        System.out.println(matriz[i][j]);
    }
}
// Método 2: Desconociendo el número de filas y columnas.
System.out.println("\nMétodo 2");
for(int i = 0; i < matriz.length; i++){
    for(int j = 0; j < matriz[i].length; j ++){
        System.out.println(matriz[i][j]);
    }
}

// Método 3: Usando un foreach
System.out.println("\nMétodo 3");
for (int[] matriz1 : matriz) { // Elimina una dimensión
    for(int matriz11: matriz1){ // Sólo 
        System.out.println(matriz11);
    }
}

    EJERCICIO: Arreglo multidimensional.

    • utilizando un arreglo multidimensional, escribir un programa en java, que imprima la siguiente información de una matriz.
            Java    Javascript  Bases de datos
        Hugo    Pedro       Victor
        Paco    Ana         Elena
        María   Juan        María

    package ejemplos;

public class Ejemplo {

    // Método principal
    public static void main(String[] args) {
        String[][] nombres = {
            {"Java", "Javascript", "Base de datos"},
            {"Hugo", "pedro", "Victor"},
            {"Paco", "Ana", "Elena"},
            {"María", "Juán", "María"},
        };

        for (int i = 0; i < nombres.length; i++) {
            for (int j = 0; j < nombres[i].length; j++) {
                if(nombres[i][j].equals("Javascript")){
                    System.out.print(nombres[i][j] + "\t");
                }else{
                    System.out.print(nombres[i][j] + "\t\t");
                }

            }
            System.out.println("");
        }
    }
}
