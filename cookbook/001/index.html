    Agregar apache tomcat a eclipse

    En el siguiente tutorial se muestra como poder agregar Apache tomcat a eclipse.

    Descargar tomcat: https://tomcat.apache.org/download-80.cgi

    Al descomprimir, la carpeta se debe llevar a un lugar en el que lo podamos encontrar, para el caso de MAC, este puee ser llevado a la siguiente ruta:

    /Applications/contenedor/apache-tomcat-9.0.31

    De esta forma ya tenemos el servidor en nuestra máquina, ahora queda agregarlo a eclipse.

    Accedemos con ⌘, o vamos a Eclipse -> preferences y lo agregamos.

    Si descargamos la versión 9.* podemos seleccionar la siguiente opción:

    En esta ventana, debemos buscar la carpeta donde se encuentra el direcctorio de apache que descargamos.

    Para probar iniciamos un proyecto.

    Entramos a las propiedades del proyecto y agregamos una nueva librería en la pestaña Libraries.

    Finalmente, creamos un index.jsp, y probamos ejecutarlo con el servidor.

    Con esto ya tenemos integrado el servidor de aplicaciones web de Apache Tomcat.
